## What It Does

Bird Language ML listens to bird audio recordings and classifies both the **species** and the **communication type** ‚ÄĒ telling you not just *which* bird is singing, but *why*.

---

## Architecture

```
Audio File (.wav / .mp3)
        ‚Ēā
        ‚Ėľ
   Sliding Window (5s chunks, 50% overlap)
        ‚Ēā
        ‚Ėľ
   Mel-Spectrogram (128 mel √ó 313 time frames)
   + SpecAugment (training only)
        ‚Ēā
        ‚Ėľ
   EfficientNet-B0 Backbone (ImageNet pretrained)
   Global Average Pooling ‚Üí 1280-dim embedding
        ‚Ēā
        ‚Ēú‚ĒÄ‚ĒÄ‚Ėļ Species Head     ‚Üí 50 species classes
        ‚ĒĒ‚ĒÄ‚ĒÄ‚Ėļ Call Type Head   ‚Üí 9 communication categories
```

## ūüĒä Call Type Categories

| Label | Meaning |
|----|---------|
| `alarm_call` | Predator warning, recruits mob response |
| `mating_song` | Mate attraction, fitness advertisement |
| `contact_call` | Flock cohesion while moving/foraging |
| `territorial_call` | Boundary defense against rivals |
| `distress_call` | Captured/cornered, recruits helpers |
| `begging_call` | Juveniles demanding food |
| `flight_call` | Migration contact, nocturnal navigation |
| `foraging_sound` | Food search, food discovery signal |
| `dawn_chorus` | Multi-function sunrise singing |

## Step 2 ‚ÄĒ Train

```bash
python -c "from src.train import train; train()"
```

Key training features:
- **EfficientNet-B0** pretrained backbone (10√ó faster convergence vs random init)
- **Differential learning rates** ‚ÄĒ backbone at 0.1√ó LR, heads at 1√ó LR
- **Mixed precision (AMP)** ‚ÄĒ 2√ó speedup on CUDA
- **SpecAugment** ‚ÄĒ frequency/time masking for regularization
- **Mixup** ‚ÄĒ interpolation between spectrograms for robustness
- **Cosine annealing with warmup** ‚ÄĒ stable convergence
- **Class-balanced sampling** ‚ÄĒ handles rare species
- **Early stopping** ‚ÄĒ prevents overfitting

Expected training time:
| Hardware | Time |
|---|---|
| GPU V100/A100 | ~2 hours / 50 epochs |
| GPU RTX 3080 | ~4 hours |
| CPU only | ~24 hours *(not recommended)* |
